
# Java Chat Application - Registration & Login System

A simple Java Swing-based chat application that allows users to register, log in, and validate input.


## Features

- User registration with validation
- Login functionality
- Input validation for username, password, and phone number
- Converts phone number to international format


## Technologies

- Java 17

- Swing (JOptionPane)

- JUnit 5 for unit testing

- NetBeans or IntelliJ IDEA for development


## Installation

- Open the project in your preferred IDE (NetBeans or IntelliJ).

- Run Main.java to start the app.

- Use MainTest.java for running unit tests.
    
## Running Tests

**Unit tests for the core logic of the application are implemented in the MainTest.java class using JUnit 5.**

- Test Coverage:
Username Validation: Ensures that usernames are at least 6 characters long and contain only letters.

- Password Validation: Checks that passwords are at least 8 characters long.

- Phone Number Validation: Verifies that the phone number matches the South African format (0XXXXXXXXX).

- Phone Number Format Conversion: Confirms that the phone number is correctly converted to the international format (+27XXXXXXXXX).

- Tests can be run in your IDE using the "Test File" option or via the terminal.
